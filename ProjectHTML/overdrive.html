OverDrive

The project I am currently working on is a personal project being made with Unity. It is a music/rhythm game where the player controls a car driving down the streets of Miami, and can collect notes to the beat of the music to increase their score

Download the current build!

I created scripts using C# for audio visualization, which add effect to the game by making the buildings bounce up and down, and the screen shake in rhythm with the music that's being played

I started by using Unity's API to receive an array of frequency data corresponding to the current audio source that is updated every frame. Using this data I was able to create a basic visualization structure:

After sorting through the data and dividing the values into the 8 frequency bands, it becomes much easier to deal with and visualize. The values farthest to the right correspond to the deeper bass frequencies, while the values to the left are the higher pitched frequencies. Using this data I was able to manipulate other in-game data to react to the music such as textures, lights, and scales.

I also modeled and textured the 3D assets used in the game, including the car that the player controls. The models were created in Blender and textured in Substance Painter

Using the points that are accumulated from levels, the player will be able to purchase skins for their car for customization
