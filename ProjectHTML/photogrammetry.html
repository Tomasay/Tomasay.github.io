Photogrammetry

For this project I created an optimized model of a gamecube controller using photogrammetry. I used Agisoft Metashape to generate a model from a series of pictures, then retopologized the model in Maya. Once I had a clean, UV'd model I reprojected the generated textures and cleaned them up for the final result
